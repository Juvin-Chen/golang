# Go Stage1 Basics（Go 的语言基础）

本目录是 Go 语言基础阶段的学习内容，包含**章节源码、实验、知识点文档和项目实践**。

------

## 目录结构说明

### 1. `src/` - 章节源码

- 每个子目录对应一个核心章节（如 `chapter1_基础语法`、`chapter2_流程控制与函数`）
- 包含 `chapterX.go`（知识点代码示例）和 `chapterX.md`（章节知识点梳理）
- 与 `lab/` 中的实验一一对应，是实验的基础代码参考

### 2. `lab/` - 实验

- 每个子目录对应一个章节的实验（如 `lab1_基础语法`、`lab2_流程控制与函数` 等）
- 包含 `labX.md`（实验要求 + 步骤）和 `labX.go`（实验代码）
- 与 `src/` 中的章节一一对应，用于巩固知识点

### 3. `docs/` - 知识点文档

- 按知识点拆分的 Markdown 文档（如编码、包机制、值类型等）
- 整理基础语法的核心概念、规则和细节，是理解 `src/` 和完成 `lab` 的辅助资料

### 4. `projects/` - 项目实践

- 存放 Go 语言基础阶段的小型项目
- 用于综合运用所学知识点，提升工程化能力

### 5. `Go_Primer_Book.md` - 书籍

- 整合所有章节知识点的完整书籍式文档
- 按章节系统梳理 Go 语言基础，适合作为学习和复习的核心资料
